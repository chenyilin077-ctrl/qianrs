# 二维云台 — 硬件架构与通信链路

这是本项目最核心的通信技术——利用 UART 空闲中断实现变长帧的零拷贝接收。

### 原理

```
DMA 持续接收:
Byte0 Byte1 Byte2 ... ByteN  [空闲>1字节时间]
|----- DMA自动搬运 -----|     |-- IDLE中断触发 --|
```

当 UART 线上超过 1 个字节时间没有新数据，硬件自动置位 IDLE 标志 → 触发中断。

### USART1 中断处理（电机应答）

```c
void USART1_IRQHandler(void) {
    if(__HAL_UART_GET_FLAG(&huart1, UART_FLAG_IDLE) != RESET) {
        __HAL_UART_CLEAR_IDLEFLAG(&huart1);       // 1. 清IDLE标志
        HAL_UART_DMAStop(&huart1);                // 2. 暂停DMA
        rxCount = CMD_LEN - __HAL_DMA_GET_COUNTER(&hdma_usart1_rx); // 3. 计算实际字节数
        rxFrameFlag = true;                        // 4. 通知主循环
        HAL_UART_Receive_DMA(&huart1, rxCmd, CMD_LEN); // 5. 重启DMA
    }
    HAL_UART_IRQHandler(&huart1);
}
```

### USART2 中断处理（K230 坐标）

```c
void USART2_IRQHandler(void) {
    if(__HAL_UART_GET_FLAG(&huart2, UART_FLAG_IDLE) != RESET) {
        // 1. 硬件清IDLE（读SR+DR）
        volatile uint32_t temp;
        temp = huart2.Instance->SR;
        temp = huart2.Instance->DR;

        // 2. 关DMA
        __HAL_DMA_DISABLE(huart2.hdmarx);

        // 3. 计算接收字节数
        k230_rx_count = K230_CMD_LEN - __HAL_DMA_GET_COUNTER(huart2.hdmarx);

        // 4. 通知主循环
        k230_rx_flag = true;

        // 5. 手动重置DMA计数器并重启（非HAL方式，更可靠）
        __HAL_DMA_SET_COUNTER(huart2.hdmarx, K230_CMD_LEN);
        __HAL_DMA_ENABLE(huart2.hdmarx);

        return; // 跳过HAL_UART_IRQHandler
    }
    HAL_UART_IRQHandler(&huart2);
}
```

> **USART1 vs USART2 的差异**: USART1 用 `HAL_UART_DMAStop` + `HAL_UART_Receive_DMA`，USART2 用手动 `__HAL_DMA_DISABLE` + `__HAL_DMA_SET_COUNTER` + `__HAL_DMA_ENABLE`。后者更底层，适合高频小包场景，避免 HAL 库的开销。

## 视觉追踪控制回路

### K230 帧解析

```c
void Parse_K230_Black_Target(uint8_t *pBuffer, uint16_t len) {
    for (i = 0; i < len - 6; i++) {
        // 严格校验帧头帧尾
        if (pBuffer[i]==0xAA && pBuffer[i+1]==0x55 && pBuffer[i+6]==0xBB) {
            int16_t target_x = (pBuffer[i+2]<<8) | pBuffer[i+3];
            int16_t target_y = (pBuffer[i+4]<<8) | pBuffer[i+5];

            // 计算像素偏差
            float error_x = 320.0f - (float)target_x;  // 图像中心 - 目标
            float error_y = 240.0f - (float)target_y;
```

### P 控制器

```c
// 速度 = 像素偏差 x KP
float vel_yaw   = error_x * 0.15f;   // YAW: 水平方向
float vel_pitch = error_y * 0.15f;   // PITCH: 垂直方向

// 方向判断
dir_yaw   = (vel_yaw   >= 0) ? 0 : 1;  // 0=正向, 1=反向
dir_pitch = (vel_pitch >= 0) ? 0 : 1;

// 速度限幅
if (vel_yaw   > 300.0f) vel_yaw   = 300.0f;  // YAW max
if (vel_pitch > 200.0f) vel_pitch = 200.0f;  // PITCH max

// 死区 (10像素内不动作)
if (fabs(error_x) <= 10.0f) { vel_yaw = 0; dir_yaw = 0; }
if (fabs(error_y) <= 10.0f) { vel_pitch = 0; dir_pitch = 0; }
```

### 电机速度指令

```c
// 串行发射，无冲突
X_V2_Vel_Control(MOTOR_YAW_ID,   dir_yaw,   ACC_YAW,   vel_yaw,   false);
while(huart1.gState != HAL_UART_STATE_READY);  // 等DMA发完
HAL_Delay(5);

X_V2_Vel_Control(MOTOR_PITCH_ID, dir_pitch, ACC_PITCH, vel_pitch, false);
while(huart1.gState != HAL_UART_STATE_READY);
HAL_Delay(5);
```

### 主循环调度

```c
while (1) {
    // 收到K230新数据 且 距上次控制 >= 30ms
    if (k230_rx_flag && (HAL_GetTick() - last_track_time >= 30)) {
        k230_rx_flag = false;
        last_track_time = HAL_GetTick();

        Parse_K230_Black_Target(k230_rx_buf, k230_rx_count);

        // 重启DMA接收
        __HAL_UART_CLEAR_OREFLAG(&huart2);
        HAL_UART_Receive_DMA(&huart2, k230_rx_buf, K230_CMD_LEN);
    }

    // 防抖：30ms内重复数据丢弃
    if (k230_rx_flag && (HAL_GetTick() - last_track_time < 30)) {
        k230_rx_flag = false;
    }
}
```

## 上电初始化流程

```
上电 → HAL_Init → 时钟180MHz → GPIO/DMA/USART1/USART2初始化
→ USART1 IDLE中断使能 + DMA接收启动
→ Gimbal_Init_Zero():
    1. HAL_Delay(1000)           ← 等电机驱动器自己开机
    2. X_V2_En_Control(YAW)      ← 使能YAW电机
    3. while(huart1不空闲);        ← 等DMA发送完成
    4. HAL_Delay(50)
    5. X_V2_En_Control(PITCH)    ← 使能PITCH电机
    6. HAL_Delay(100)
    7. X_V2_Reset_CurPos_To_Zero(YAW)    ← 当前位置归零
    8. HAL_Delay(500)                    ← 等Flash写入
    9. X_V2_Reset_CurPos_To_Zero(PITCH)  ← 当前位置归零
    10. HAL_Delay(500)
→ USART2 IDLE中断使能 + DMA接收启动
→ 进入主循环追踪
```

> 归零后的 500ms 死等是关键——电机内部写 Flash 存储零点，期间不能被打扰。

## X_V2 电机驱动库概览

张大头闭环步进驱动器的完整指令集：

| 类别 | 函数 | 说明 |
|------|------|------|
| 触发动作 | X_V2_Trig_Encoder_Cal | 编码器校准 |
| 触发动作 | X_V2_Reset_Motor | 重启电机 |
| 触发动作 | X_V2_Reset_CurPos_To_Zero | 当前位置清零 |
| 运动控制 | X_V2_En_Control | 使能/失能 |
| 运动控制 | X_V2_Vel_Control | 速度模式 |
| 运动控制 | X_V2_Traj_Pos_Control | 梯形加减速位置模式 |
| 运动控制 | X_V2_Torque_Control | 力矩模式 |
| 云台封装 | Gimbal_Init | 初始化+使能+归零 |
| 云台封装 | Gimbal_Set_Angle | 绝对角度控制 (0~360度) |

每条指令通过 `HAL_UART_Transmit_DMA(&huart1, cmd, 4)` 异步发送，发送后需等待 `huart1.gState == HAL_UART_STATE_READY` 确保 DMA 传输完成。
