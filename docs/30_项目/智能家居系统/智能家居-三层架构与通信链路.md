# 智能家居 — 三层架构与通信链路

## 架构总览

```
[语音输入] → AGX Orin → WiFi → STM32F429 → WiFi → ESP32×3 → [家电执行]
               ↑                    ↑
          Whisper+Mistral    FreeRTOS+TouchGFX
                             BME280+DHT11+舵机+GPIO
```

## Tier 1: AGX Orin — AI 大脑

| 组件 | 功能 | 说明 |
|------|------|------|
| Whisper | 语音→文字 | 本地推理, 低延迟 |
| Mistral 3 14B | 语义理解 | 解析意图: 设备+动作+参数 |
| 输出 | JSON 指令 | `{"action":"open","device":"空调"}` |

> 指令经 WiFi 发送到 ESP8266 的 TCP Server (端口 8080)

## Tier 2: STM32F429 — 中央网关

### ESP8266 通信 (USART3)

```
PB10(TX) / PB11(RX) @ 115200
└── AT 指令集 (ATK_MW8266D 驱动)
```

**WiFi 配置**:

```c
// 1. 模块就绪等待
Soft_Delay_ms(2000);

// 2. AT 通信检测
Send_AT_Command_Ext("AT\r\n", 1000, 0);

// 3. Station 模式
Send_AT_Command_Ext("AT+CWMODE=1\r\n", 2000, 0);

// 4. 连接路由器 (带重试)
Send_AT_Command_Ext("AT+CWJAP=\"zhinengjiaju\",\"12345678\"\r\n", 10000, 0);

// 5. 多连接模式
Send_AT_Command_Ext("AT+CIPMUX=1\r\n", 2000, 0);

// 6. 启动 TCP Server
Send_AT_Command_Ext("AT+CIPSERVER=1,8080\r\n", 2000, 0);
```

### UART3 中断解析

```c
void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart) {
    if (huart->Instance == USART3) {
        // 逐字节接收, 构建 rx_buffer

        // 检测 OK (AT应答)
        if (最后两字节 == "OK") is_ok_received = 1;

        // 检测 BME 传感器数据
        if (strstr(rx_buffer, "BME:T=")) Parse_BME_Data(rx_buffer);

        // 检测 AI JSON 指令 (以 } 结尾)
        if (rx_byte == '}') {
            osMessagePut(aiCommandQueueHandle, ai_cmd_buffer, 0);
        }

        // 重新使能中断接收下一字节
        HAL_UART_Receive_IT(&huart3, &rx_byte, 1);
    }
}
```

### FreeRTOS 双队列架构

```
[UART3中断] ──→ aiCommandQueueHandle ──→ [TouchGFX Task]
                                              ↓
                                          解析指令
                                          控制GPIO/PWM
                                              ↓
[TouchGFX Task] ──→ wifiCommandQueueHandle ──→ [WiFi Send Task]
                                                   ↓
                                            ESP8266_Broadcast()
```

**WiFi 发送任务（非阻塞）**:

```c
void StartWifiSendTask(void const * argument) {
    for(;;) {
        evt = osMessageGet(wifiCommandQueueHandle, osWaitForever);
        if (evt.status == osEventMessage) {
            char* payload = (char*)evt.value.p;
            ESP8266_Broadcast(payload);  // 发送到 ESP32 #0,1,2
        }
    }
}
```

**广播函数**:

```c
void ESP8266_Broadcast(char* payload) {
    for (uint8_t i = 0; i < 3; i++) {
        ESP8266_Send_To_ID(i, payload);
    }
}
```

### 传感器采集

```c
// BME280 数据格式
"BME:T=25.3H=65.2P=1013.2"

void Parse_BME_Data(char *rx_buffer) {
    char *t_pos = strstr(rx_buffer, "T=");
    char *h_pos = strstr(rx_buffer, "H=");
    char *p_pos = strstr(rx_buffer, "P=");

    if (t_pos && h_pos && p_pos) {
        current_temp  = atof(t_pos + 2);
        current_hum   = atof(h_pos + 2);
        current_press = atof(p_pos + 2);
    }
}
```

### 执行器控制

| 设备 | 控制方式 | 代码 |
|------|----------|------|
| 电视 | GPIO PB0 | `HAL_GPIO_WritePin(GPIOB, GPIO_PIN_0, RESET/SET)` |
| 监控 | GPIO PB1 | `HAL_GPIO_WritePin(GPIOB, GPIO_PIN_1, SET)` |
| 空调开关 | GPIO PA4 | `HAL_GPIO_WritePin(GPIOA, GPIO_PIN_4, SET/RESET)` |
| 制冷/热 | GPIO PA5 | `HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, SET/RESET)` |
| 风扇 PWM | TIM1 CH2 | `__HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_2, duty)` |
| 风口舵机 | TIM2 CH3 | `__HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_3, angle)` |

**舵机控制**:

```c
// 0~180° → 500~2500us
void Servo_SetAngle(uint16_t angle) {
    if (angle > 180) angle = 180;
    uint32_t ccr_val = 500 + (angle * 2000 / 180);
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_3, ccr_val);
}
// TIM2: 90分频 → 2MHz, 周期19999 → 50Hz (20ms标准舵机周期)
```

### TouchGFX AI 指令匹配

```c
void Screen2View::handleTickEvent() {
    // 每60 ticks(~1s) 刷新传感器显示
    if (++tickCounter >= 60) {
        updateSensorData(current_temp, current_hum, current_press);
    }

    // 从队列取 AI 指令
    evt = osMessageGet(aiCommandQueueHandle, 0);
    if (evt.status == osEventMessage) {
        char* raw = (char*)evt.value.p;

        // UTF-8 关键词匹配
        if (strstr(raw, "空调")) {     // \xE7\xA9\xBA\xE8\xB0\x83
            if (strstr(raw, "开"))     // \xE5\xBC\x80
                open_kongtiao();        // 开空调
            if (strstr(raw, "制热"))   // \xE5\x88\xB6\xE7\x83\xAD
                cold_hot();             // 切换制热
        }
        else if (strstr(raw, "电视"))  opentv();
        else if (strstr(raw, "窗帘"))  oppo();
    }
}
```

## Tier 3: ESP32 — 边缘执行

3 个 ESP32 节点, 各管一个家电:

| ESP32 ID | 设备 | 接收指令 | 执行 |
|----------|------|----------|------|
| #0 | 电视 | `OPENTV_ON` / `OPENTV_OFF` | 红外/继电器 |
| #1 | 监控 | `OPENJK_ON` | 继电器开关 |
| #2 | 窗帘 | `OPENT_ON` / `OPENT_OFF` | 电机正反转 |

> ESP32 代码不在本工程中, 仅通过 WiFi 接收广播文本指令。

| 定时器 | 功能 | 配置 |

| TIM1 CH2 | 风扇 PWM | 180分频→1MHz, 周期999→1kHz |
| TIM2 CH3 | 舵机 PWM | 90分频→2MHz, 周期19999→50Hz |
| TIM9 | 通用 | 90分频→2MHz, 周期999→2kHz |
