# 李萨茹图形测量仪 — 项目总览

## 比赛背景

- **赛事**: 第十九届电子设计竞赛 — D题
- **主控芯片**: STM32F429IGT6（阿波罗 F429 开发板）
- **主频**: 180MHz（HSE 25MHz → PLL: M=25, N=360, P=2）
- **开发环境**: Keil MDK-ARM + CubeMX，HAL 库

## 核心功能

| 功能 | 实现方式 | 说明 |
|------|----------|------|
| 双路频率测量 | TIM1 输入捕获（PA8/PA9） | 180MHz时钟，0预分频，下降沿捕获，周期法测频 |
| 相位差测量 | TIM1 CH1/CH2 时差计算 | 16位自然溢出 + 取模运算，消除大角度跳变 |
| 李萨茹图形渲染 | 参数方程数学解算 | 240点 sin() 曲线，FMC SDRAM 高速绘图 |
| 数字移相网络 | MCP41010 双路 SPI + X9313 | 手测校准表 + 线性插值，5°~180° 精准控制 |
| 触控交互 | FT5206/GT9xxx 触控屏 | 4x4 数字键盘 + RUN/STOP + 相位输入 |

## 引脚分配表

| 引脚 | 功能 | 所属模块 |
|------|------|----------|
| PA8 | TIM1_CH1 | 外部信号X 频率捕获 |
| PA9 | TIM1_CH2 | 外部信号Y 频率捕获 |
| PA6 | SPI1_MISO | MCP41010 数据读取 |
| PA7 | SPI1_MOSI | MCP41010 数据写入 |
| PB3 | SPI1_SCK | MCP41010 时钟 (90MHz) |
| PA4 | GPIO OUT (CS1) | MCP41010 U1 片选 |
| PA3 | GPIO OUT (CS2) | MCP41010 U2 片选 |
| PA5 | GPIO OUT (CS) | X9313 片选 |
| PB1 | GPIO OUT (UD) | X9313 方向控制 |
| PB0 | GPIO OUT (INC) | X9313 脉冲输入 |
| PE5/PE6 | GPIO OUT | 输出控制 |

## 时钟与定时器

| 定时器 | 时钟源 | 预分频 | 周期 | 中断频率 | 用途 |
|--------|--------|--------|------|----------|------|
| TIM1 | 180MHz(APB2) | 0 | 65535 | 事件触发 | 双路输入捕获测频 |
| TIM2 | 180MHz→2MHz(APB1) | 89 | 999 | 2kHz | 触控扫描 |
| TIM3 | 180MHz→20kHz(APB1) | 8999 | 9999 | ~2Hz | 图形刷新 |

## 关键驱动文件

| 文件 | 说明 |
|------|------|
| Core/Src/tim.c | TIM1/2/3 配置 |
| Core/Src/main.c | 主程序、TIM1捕获回调、频率滤波 |
| BSP/RGBLCD/cm.c | 李萨茹解算 + UI 交互 + 触控键盘 |
| BSP/MCP410XX/MCP41xx.c | MCP41010 SPI 驱动 + 手测校准表 + 插值 |
| BSP/f.c | X9313 3线数字电位器驱动 |
| Core/Src/spi.c | SPI1 硬件配置 (90MHz) |
| BSP/LCD/ | LTDC LCD 驱动 + SDRAM |
