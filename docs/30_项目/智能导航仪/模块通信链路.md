# 模块通信链路




### 软件 I2C 实现对比

| 特性 | I2C-1 (myiic.c) | I2C-2 (paj7620u2_iic.c) |
|------|-----------------|--------------------------|
| SCL 引脚 | PH4 | PA4 |
| SDA 引脚 | PH5 | PA11 |
| 延时 | delay_us(2) | delay_us(10) |
| 约 SCL 频率 | ~100kHz | ~50kHz |
| 从设备 | PCF8574 | PAJ7620U2 |
| ACK 等待 | 有 | 有 |
| 起始/停止 | 标准 | 标准 |

### I2C 时序参数

`
标准 I2C 时序（100kHz）:
  t_HD;STA (START hold)  ≥ 4.0μs
  t_LOW   (SCL low)      ≥ 4.7μs
  t_HIGH  (SCL high)     ≥ 4.0μs
  t_SU;STO (STOP setup)  ≥ 4.0μs
  t_BUF   (bus free)     ≥ 4.7μs

本项目软件实现:
  delay_us(2):  理论 2μs, 实测 ~5μs (含函数开销)
  delay_us(10): 理论 10μs, 实测 ~13μs (含函数开销)
`

## UART 通信链路

### 串口分配

| 串口 | 引脚 | 波特率 | 特殊配置 | 对接模块 |
|------|------|--------|----------|----------|
| USART1 | PA9(TX)/PA10(RX) | 115200 | 标准 | 调试/日志 |
| USART2 | PA2(TX)/PA3(RX) | 115200 | 标准 | 模块通信 |
| USART6 | PC6(TX)/PC7(RX) | 115200 | 接收触发PE5/PE6 | 模块通信 |
| UART5 | PC12(TX)/PD2(RX) | 115200 | DMA发送 | 模块通信 |

### USART6 特殊逻辑

USART6 接收特定数据后触发 GPIO 动作：

`c
// 当 USART6 接收到 0x01
// → PE5, PE6 输出高电平
// 用于联动控制外部设备
`

### 数据流示意

`
语音指令流:
  ASRPRO → USART1 RX → main.c 解析 → 执行动作

手势数据流:
  PAJ7620U2 → I2C-2 → paj7620u2.c → main.c 解析 → 执行动作 + PB15输出

视觉数据流:
  K230 → UART5 RX(DMA) → main.c 解析 → 显示结果

温湿度数据流:
  DHT11 → 单总线 PC10 → DHT11.C → main.c → 显示结果
`

## GPIO 直连控制

| 引脚 | 方向 | 触发条件 | 功能 |
|------|------|----------|------|
| PE5 | OUT | USART6 收到 0x01 | 外部设备控制 |
| PE6 | OUT | USART6 收到 0x01 | 外部设备控制 |
| PB15 | OUT | GES_WAVE 手势 | 手势反馈输出 |

## 中断与 DMA 分配

| 资源 | 用途 | 优先级 |
|------|------|--------|
| SysTick | 系统滴答 1ms | 最低 |
| USART1 IRQ | 语音数据接收 | 中 |
| USART2 IRQ | 模块通信接收 | 中 |
| USART6 IRQ | 模块通信接收 | 中 |
| UART5 DMA TX | 大数据量发送 | - |
| TIM6 | DHT11 微秒延时 | 不使用中断 |
| TIM13 | PWM 输出 | - |

## 通信优先级与调度

`
主循环调度:
  1. DHT11 温湿度读取 (25ms, 关中断)
  2. PAJ7620U2 手势查询 (快速 I2C 读)
  3. 触控坐标读取
  4. UART 数据解析与动作执行
  5. UI 刷新
`

DHT11 读取期间关中断保护时序，读取完成后恢复中断处理 UART 数据。
