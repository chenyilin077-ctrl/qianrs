## 技术栈

- **MCU**: STM32F429IGT6 (180MHz Cortex-M4)
- **OS**: FreeRTOS
- **UI**: TouchGFX / LCD
- **AI**: Whisper + Mistral 3 14B (AGX Orin)
- **传感器**: DHT11, BME280, PAJ7620U2, K230
- **通信**: UART, I2C, SPI, WiFi (ESP8266), DMA
- **控制**: 步进电机, 舵机, 数字电位器, GPIO
